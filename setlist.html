This file is self-contained: songs and all your per-song settings save into setlist.html itself. Copy or send the one file to any device and it opens ready to go. The first Ctrl+S each session asks where to save — pick this same file to update it in place. On browsers without direct file saving, Save downloads a fresh copy instead; replace the old file with it. On phones and tablets: swipe left/right or tap the edge arrows to change songs. For a full-screen view, tap ⛶ (Android and desktop). On iPhone, Safari doesn't allow true full screen — instead use Share → Add to Home Screen, then launch from that icon to get the whole screen with no browser bars. Tip: press F11 for full screen during worship.
